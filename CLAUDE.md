## 🎯 프로젝트 목적

공대 학술 동아리/학과 구성원들이 좋은 유튜브 영상, 논문, 아티클을 **부담 없이 공유하고** 자연스럽게 서로 자극받는 **학술 피드 커뮤니티**.

### 해결하는 문제
- 단톡방에 링크 던지면 → 바로 묻힘
- 인스타에 올리기엔 → 너무 학술적
- 노션 정리는 → 너무 무거움
- **마찰 없는 공유 공간이 없다**

## 🗂️ DB 스키마

```ts
Post {
  id          : UUID         // 게시물 고유 번호
  url         : String       // 유튜브 / 논문 / 아티클 링크
  title       : String       // 링크에서 자동 추출된 제목

  author      : User         // 작성자 이름 + 전공
  created_at  : Timestamp
}
```

## ✨ 핵심 기능

### ① 초간편 공유 (Low-Friction Post)
- URL 붙여넣기 → 제목/썸네일 자동 추출 (Link Auto-Preview)
- 한 줄 평만 써도 포스팅 완료
- 카테고리 태그 1개 선택: `[로보틱스]` `[AI]` `[화생공]` `[미래사회]`

### ② 큐레이션 피드 (The Stream)
- **Mixed Feed**: 전체 과 트렌드 섞인 메인 홈
- **Filter Tabs**: 상단 탭으로 특정 카테고리만 필터링
- **Reaction Only**: 댓글 대신 이모지 반응 (`오..` / `대박` / `유익함`)

### ③ 통학 스낵 모드 (Commute Mode)
- 오늘의 추천 카드 슬라이드 (알고리즘 아닌 **지인 추천**)
- Save for Later 북마크 (나중에 데스크탑에서 제대로 보기)
