# InsightStream_

> "공유는 가볍게, 인사이트는 묵직하게."

공대 학술 동아리/학과 구성원들이 유튜브 영상, 논문, 아티클을 부담 없이 공유하고 서로 자극받는 학술 피드 커뮤니티.

---

## 실행 방법

### 1. 사전 준비

- **Node.js 18** 이상 설치 필요 ([https://nodejs.org](https://nodejs.org))

### 2. 의존성 설치

```bash
cd InsightStream
npm install
```

### 3. 개발 서버 실행

```bash
npm run dev
```

실행 후 브라우저에서 **http://localhost:3000** 접속.

### 4. 프로덕션 빌드 (선택)

```bash
npm run build
npm run start
```

## 사용 가이드

### 피드 둘러보기

- 메인 화면에 공유된 글들이 카드 형태로 표시됩니다.
- 상단 **필터 탭**으로 카테고리별 필터링 가능:
  - `전체` · `로보틱스` · `AI` · `화생공` · `미래사회` · `저장됨`

### 글 상세 보기

- **카드를 클릭**하면 상세 모달이 열립니다.
- 모달 구성:
  - 작성자 정보 + 카테고리 + 날짜
  - 작성자의 글 전문
  - YouTube 영상은 **iframe으로 바로 재생** 가능
  - 비YouTube 링크는 **외부 링크 카드**로 표시
  - 리액션 버튼 + 댓글 목록
- 모달 닫기: `← 뒤로` 버튼 / 배경 클릭 / `ESC` 키

### 인사이트 공유하기

1. 상단의 **"+ 인사이트 공유하기"** 버튼 클릭
2. URL 붙여넣기 → 포커스 해제 시 제목/썸네일 자동 추출
3. 텍스트 영역에 생각을 자유롭게 작성 (여러 문단 가능)
4. 이름, 전공 입력
5. 카테고리 선택 후 **"공유하기"** 클릭

### 리액션

- 카드 하단 또는 상세 모달에서 리액션 버튼 클릭:
  - 👀 `오..` · 🤯 `대박` · 📚 `유익함`

### 댓글 달기

1. 카드 클릭 → 상세 모달 열기
2. 하단 댓글 입력 폼에 이름, 전공, 댓글 내용 작성
3. **"작성"** 버튼 클릭 → 즉시 반영

### 북마크 (Save for Later)

- 카드 우측 상단의 **북마크 아이콘** 클릭하여 저장/해제
- 필터 탭에서 **"저장됨"** 선택하면 북마크한 글만 모아보기
- 브라우저 로컬 스토리지에 저장되므로 새로고침해도 유지

## 프로젝트 구조

```
src/
├── app/
│   ├── page.tsx            # 메인 페이지
│   ├── layout.tsx          # 루트 레이아웃
│   ├── globals.css         # 글로벌 스타일 + 디자인 토큰
│   └── api/og/route.ts     # OG 메타데이터 추출 API
├── components/
│   ├── MainFeed.tsx        # 상태 관리 (피드, 모달, 댓글)
│   ├── FilterTabs.tsx      # 카테고리 필터 탭
│   ├── PostForm.tsx        # 글 작성 폼
│   ├── Feed.tsx            # 피드 그리드
│   ├── FeedCard.tsx        # 개별 카드
│   ├── PostDetail.tsx      # 상세 모달 (댓글, 임베드)
│   ├── ReactionBar.tsx     # 리액션 버튼
│   └── BookmarkButton.tsx  # 북마크 버튼
└── lib/
    ├── types.ts            # 타입 정의 (Post, Comment, etc.)
    ├── categories.ts       # 카테고리 설정
    └── dummy-data.ts       # 더미 데이터
```
